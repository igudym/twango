twango by dagray

A series of tweaks to the django framework

An open source set of extensions to make setting up new django projects go more smoothly while still retaining the flexability of the django framework. It just provides a set of templates that allow you to really dive in and get started on the main project rather than repeat the tasks you often need to perform setting up a new django project.

Dependencies

pip, python, django

Install

sudo pip install https://github.com/dagray/twango/tarball/master twango-admin.py startproject myproject source bootstrap.sh
